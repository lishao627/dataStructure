# 🧮 长整数处理系统 (Big Integer Processing System)

## 📖 项目简介
本项目是一个用于处理超长整数（超过编程语言原生整型范围）的控制台程序。  
它通过链表或数组的方式实现大数的存储与运算，能够支持 100 位甚至更长的整数计算。  
主要用于学习数据结构中“链表”与“栈”等知识的实际应用。

---

## 🚀 功能特性
- ✅ 支持任意长度整数的输入与存储  
- ➕ 实现了基本的**加法、减法、2机制以及十六进制转换**运算  
- 🔁 支持从文件读取或写入结果  
- 🧩 使用**结构体与指针**管理内存，实现灵活的运算逻辑  
- 💬 提供清晰的用户操作菜单与输入提示  

---

## 🧠 技术实现
- **语言**：C / C++  
- **开发环境**：Visual Studio 2022  
- **主要数据结构**：
  - 链表（用于存储长整数的每一位）
  - 栈（辅助运算处理）
- **算法设计**：
  - 按位加减实现大数运算
  - 手动进位与借位逻辑
  - 多位乘法模拟笔算法
